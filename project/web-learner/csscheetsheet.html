CSS Cheat Sheet
1. Selectors 
* – Selects all elements.
element – Targets all elements of that tag (p → all paragraphs).
.class – Selects elements with that class name.
#id – Selects an element with that unique ID.
element, element – Applies styles to multiple selectors at once.
element element – Targets elements inside another element.
element > element – Targets direct children only.
element + element – Selects the next sibling element.
element ~ element – Selects all following siblings.
[attribute=value] – Selects elements with a specific attribute/value pair.
:hover, :focus, :active – Pseudo-classes that apply styles based on user interaction.
::before, ::after – Pseudo-elements that add virtual content before or after an element.
2. Colors 
color – Changes the text color.
background-color – Changes the background color.
rgba() – Adds color with transparency (A = alpha).
linear-gradient() – Creates a smooth gradient transition between colors.
3. Text & Font 
font-family – Sets the font style.
font-size – Controls the text size.
font-weight – Makes text boldness vary (normal, bold, 100–900).
font-style – Italicizes or obliques text.
text-align – Aligns text (left, center, right).
text-transform – Changes case (uppercase, lowercase, capitalize).
text-decoration – Adds/removes underline/line-through.
line-height – Controls spacing between lines.
letter-spacing – Adjusts space between characters.
4. Box Model 
width & height – Size of the box.
margin – Space outside the element.
padding – Space inside the element.
border – Adds an outline (style, width, color).
border-radius – Rounds the corners.
box-sizing: border-box; – Makes width include padding and border.
box-shadow – Adds shadows behind elements.
5. Display & Positioning 
display – Defines how an element is displayed (block, flex, grid, etc.).
position – Controls element’s placement (static, relative, absolute, fixed, sticky).
top, left – Used with positioning to move elements.
z-index – Controls stack order (which element is on top).
overflow – Controls content that overflows its box (hidden, scroll, auto).
6. Flexbox 
display: flex; – Enables flex layout.
flex-direction – Row or column arrangement.
justify-content – Aligns items horizontally.
align-items – Aligns items vertically.
align-content – Controls space between rows of flex items.
flex-wrap – Allows items to wrap onto multiple lines.
flex – Controls how much space each item takes.
7. Grid 
display: grid; – Enables grid layout.
grid-template-columns – Defines number and size of columns.
grid-template-rows – Defines rows.
gap – Space between rows/columns.
grid-column – Defines how many columns an element spans.
grid-row – Defines how many rows an element spans.
8. Backgrounds 
background-image – Adds a background picture.
background-repeat – Controls repetition (repeat, no-repeat).
background-size – Scales image (cover = full cover, contain = fit inside).
background-position – Controls image placement.
background-attachment – Makes background scroll with page or stay fixed.
9. Transitions & Animations 
transition – Smoothly changes properties over time.
transform – Moves, rotates, scales elements.
@keyframes – Defines animation steps.
animation – Runs the keyframe animation.
10. Miscellaneous 
opacity – Controls transparency (0 = invisible, 1 = visible).
cursor – Changes the mouse pointer (pointer, crosshair).
clip-path – Crops an element to a shape.
filter – Applies visual effects (blur, brightness, grayscale).
1. Advanced Selectors 
:nth-child(n) – Selects the nth child of a parent.
:nth-of-type(n) – Selects the nth occurrence of a specific element type.
:not(selector) – Excludes certain elements.
:first-child, :last-child – Selects the first or last child.
:checked – Matches checked checkboxes/radio buttons.
:disabled, :enabled – Styles form controls.
:empty – Selects elements with no content.
:root – Selects the root element (html).
2. Units 
Absolute units: px, pt, cm, mm, in.
Relative units: em (relative to parent font size), rem (relative to root font size), % (relative to parent),
vw (viewport width), vh (viewport height), vmin, vmax.
3. Borders & Outlines 
border-style: solid | dashed | dotted | double;
border-image: url(border.png) 30 round;
outline: 2px dashed red;
outline-offset: 5px;
4. Advanced Backgrounds 
background-clip: content-box | padding-box | border-box;
background-blend-mode: multiply | screen | overlay;
mix-blend-mode: difference | multiply;
5. Lists & Tables 
list-style-type: none | decimal | circle | square;
list-style-position: inside | outside;
border-collapse: collapse | separate;
border-spacing: 5px;
caption-side: top | bottom;
6. Responsive Design 
@media (max-width: 768px) { ... } – Media queries.
@media (prefers-color-scheme: dark) { ... } – Dark mode.
object-fit: cover | contain; – For images/videos.
aspect-ratio: 16/9; – Maintain proportion.
7. CSS Variables 
Define: :root { --main-color: #ff0000; }
Use: color: var(--main-color);
Fallback: color: var(--secondary-color, #000);
8. Advanced Flexbox 
order: 2; – Reorders items.
align-self: flex-end; – Aligns individual item differently.
flex-grow: 1; – Allows an item to grow.
flex-shrink: 0; – Prevents shrinking.
flex-basis: 200px; – Base width/height.
9. CSS Grid Advanced 
grid-auto-rows: minmax(100px, auto);
grid-auto-columns: minmax(150px, 1fr);
place-items: center; – Shortcut for align-items + justify-items.
place-content: space-around; – Controls overall alignment.
repeat(auto-fit, minmax(200px, 1fr)); – Responsive grid columns.
10. Transforms & Filters 
transform-origin: center;
perspective: 1000px; – For 3D transforms.
filter: grayscale(100%) | sepia(50%) | contrast(150%);
11. Transitions (Advanced) 
transition-property: transform, opacity;
transition-duration: 0.3s;
transition-delay: 0.2s;
transition-timing-function: ease-in-out;
12. Animations (Advanced) 
@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
  100% { transform: translateY(0); }
}
.element {
  animation: bounce 1s infinite;
}

animation-delay: 1s;
animation-fill-mode: forwards | backwards | both;
animation-iteration-count: infinite;
13. Modern CSS Features 
scroll-behavior: smooth; – Smooth scrolling.
overflow-wrap: break-word; – Break long words.
word-break: break-all; – Force breaks anywhere.
hyphens: auto; – Hyphenate long words.
backdrop-filter: blur(10px); – Glassmorphism effect.
14. Shapes & Clipping 
clip-path: polygon(50% 0, 100% 50%, 50% 100%, 0 50%);
shape-outside: circle(50%);
15. Scroll & Interaction 
overflow-x: auto; overflow-y: hidden;
scroll-snap-type: x mandatory;
scroll-margin: 20px;
pointer-events: none; – Makes element ignore clicks.
user-select: none; – Prevents text selection.
